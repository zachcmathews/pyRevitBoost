# Open Cutsheet
Launch the PDF specified in the parameter named Cutsheet.

## Instructions
1. Specify a parameter named Cutsheet of type URL and point it at a PDF.
2. Use **Open Cutsheet**.
3. Select an element to open its cutsheet in the default application for 
   viewing PDFs.
